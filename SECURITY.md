# Security Policy

Amber Lens is built trust-first: read-only, agentless, local-first, zero
telemetry, open-source.

- **Reporting:** email security@ambercloud.ai with details. Please do not open
  public issues for vulnerabilities.
- **External audit:** an independent security audit (e.g., Trail of Bits / NCC /
  Bishop Fox) is published in this repo before GA; P0/P1 findings block release.
- **SBOM:** a CycloneDX SBOM is generated per release; CI blocks GPL/AGPL
  transitive dependencies.
