# Workload Fingerprint (v1)

The Fingerprint is the versioned, anonymized output of a scan and the contract
between Lens (Foundation) and Navigator (paid). **v1 core to FREEZE by July 1.**

v1 core source workloads: **VMware vSphere · SQL Server · Oracle · OS/Host ·
Dependencies**. Fast-follow (v1.x) and later (v2) workloads attach as profiles.

Canonical JSON Schema: [`pkg/schema/fingerprint.schema.json`](../pkg/schema/fingerprint.schema.json).
Validate the Discovery Report fields against this before the freeze.
