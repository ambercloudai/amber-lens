# Contributing

Thanks for your interest in Amber Lens.

- All contributions are licensed under **Apache-2.0**.
- Collectors and profiles must be **read-only, agentless, and destination-neutral**.
- Run `make build test lint` before opening a PR.
- New workload coverage is added as an open-source **collector profile** (see
  `profiles/`), not by bloating the core.

> Governance note: until the project's Trigger Events occur, contributors engage
> under the clean-room protocol (personal hardware, independent repos, no
> employer resources) and execute a CIIAA. See the project governance docs.
