# Collector Profiles

Amber Lens grows its coverage through **open-source collector profiles** layered
over the versioned Workload Fingerprint — so new workloads slot in without
re-locking the schema.

Rules (non-negotiable):
- Every profile is **open-source** and **cross-platform / cross-cloud**.
- Profiles are **read-only and destination-neutral** — a profile may not bias the
  assessment toward any vendor or cloud.
- Vendor-curated profiles are welcome but reviewed against these rules.
